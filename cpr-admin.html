Tech Stack Guide

This admin panel currently reads from localStorage (same browser as CPR Tool).

To make it work across all users globally:

  • Host CPR Tool on Vercel (free) — gives every user the same URL
  • Connect Supabase (free tier: 50,000 users) as database
  • Replace localStorage calls with Supabase API calls
  • Admin panel reads from Supabase — see all real users
  • Cost: ₹0/month on free tiers until 50K users
